Scrolled Flow

This reading system does not support scripting. The test is not relevant, and can be marked as "null" in the report.

Tests whether setting the global rendition:flow property to scrolled-continuous results in scrollable publication in the case when scripted is supported by the reading system. The test passes if the publication is presented as one continuous scroll from spine item to spine item.

Note: Some reading systems allow readers to manually change the flow (e.g., from dynamically paginated to scrolled). Do not use these settings for this test. To pass, the reading system must adapt to the specified flow.
