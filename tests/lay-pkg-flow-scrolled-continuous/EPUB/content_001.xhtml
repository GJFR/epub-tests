Continuously-Scrolled Flow

Tests whether setting the global rendition:flow property to scrolled-continuous results in a continuously scrolling interface. The test passes if the content of this publication is presented as one continuous scrolling document (as opposed to be dynamically paginated).

Note: Some reading systems allow readers to manually change from the flow (e.g., from dynamically paginated to scrolled). Do not use these settings for this test. To pass, the reading system must adapt to the specified flow.
