Test passes (i.e., the Reading System supports playback of two consecutive XHTML Media documents, using a common MO file) if

  1. The next page is properly displayed as an XHTML content
  2. The Reading System provides the user with a way to turn text reading on
  3. If turned on, the extract on the next page is read aloud by the Reading System
  4. After the full paragraph is read, the content automatically switches to the next portion of the texts with two more paragraphs, while read aloud continues without interruption.

The Reading System may highlight the text being read with a green background with the text grayed out; but this feature is not required for the test to pass.
