rendition:align-x-center in reflowable documents

This test passes if the following page appears centered in the viewport.

This test is not applicable to reading systems that do not support rendition:align-x-center. if this reading system does not support the property, mark it as "null".

NOTE: This test may be hard to validate on reading systems offering only single-column views or portrait-only display. If the reading system does offer multiple column view or multiple orientations, ensure both settings are set to a default setting.
