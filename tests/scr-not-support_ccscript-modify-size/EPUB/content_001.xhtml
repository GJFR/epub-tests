Note that the test relies on the availability of the iframe element in HTML. If no content appears below this line in a box, or the box contains an error message, it means that the iframe feature is not supported, and the test fails.

If the iframe appears in a 200×200 square instead of a 500×70 rectangle, the container-constrained script running in an iframe could modify the dimensions of the containing rectangle. This is disallowed in EPUB 3.3, meaning that the test fails. Otherwise, the test passes.
