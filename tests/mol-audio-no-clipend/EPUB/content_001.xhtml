Test passes (i.e., the Reading System correctly handles the lack of a clipEnd attribute value) if

  1. The Reading System provides the user with a way to turn text reading on
  2. If turned on, the extract on the next page is read aloud by the Reading System
  3. The full text is read

The Reading System may highlight the text being read with a green background with the text grayed out; but this feature is not required for the test to pass. Note that, if highlighted, the highlight may get out of sync with after the second sentence: the audio keeps playing while the highlight remains on the second sentence. This is a possible effect of the missing clipEnd for the second sentence.
