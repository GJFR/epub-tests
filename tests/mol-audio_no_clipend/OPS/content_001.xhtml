Test passes (i.e., the Reading System correcty implements the lack of a clipEnd attribute value) if

  1. The Reading System provides the user with a way to turn text reading on
  2. If turned on, the extract on the next page is read aloud by the Reading System
  3. If the reading system highlights the active text, then the highlight stops at the second section
  4. The text does not stop at the end of the whole text, but continues until the end of the full audio recording.
